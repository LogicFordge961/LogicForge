Frequently Asked Questions

How do I install WatchDog?

Download the Windows executable from our downloads page, then simply double-click to run the installer. No admin rights required!

Is WatchDog free?

Yes! WatchDog is completely free and open source. We believe in making powerful tools accessible to everyone.

Does it work on macOS or Linux?

Currently, WatchDog only supports Windows. However, we're planning macOS and Linux versions for future releases.

How do I report bugs?

Please use our contact form below or open an issue on our GitHub repository.
